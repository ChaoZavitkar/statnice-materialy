Co jsou to spojové struktury

Problém s klasickými seznamy (pole)

# Klasický Python seznam (interně pole)
seznam = [10, 20, 30, 40, 50]

# V paměti: [10][20][30][40][50] - vedle sebe, souvislý blok
#            ↑   ↑   ↑   ↑   ↑
#          index 0,1,2,3,4

Problém: Co když chcete vložit prvek doprostřed?

seznam.insert(2, 25)  # Vložit 25 na pozici 2

# Musí se posunout všechny prvky doprava!
# [10][20][  ][30][40][50]  ← udělat místo
# [10][20][25][30][40][50]  ← vložit
#              ↑   ↑   ↑
#         Všechny se posunuly → O(n) - POMALÉ!

Řešení: Spojový seznam

Nápad: Prvky nemusí být vedle sebe – každý prvek "ukáže" na další.

[10|→] → [20|→] → [30|→] → [40|→] → [50|×]
  ↑
data | odkaz

Výhoda: Vložení doprostřed = jen změna odkazů! O(1)

1. Jednosměrný spojový seznam

Struktura uzlu (Node)

Každý prvek (uzel) obsahuje:

  1. Data – hodnota, kterou ukládáme
  2. Odkaz (next) – ukazatel na další uzel
class Node:
    """Jeden uzel spojového seznamu"""
    def __init__(self, data):
        self.data = data      # Uložená hodnota
        self.next = None      # Odkaz na další uzel

uzel1 = Node(10)
print(uzel1.data)   # 10
print(uzel1.next)   # None

Propojení uzlů

uzel1 = Node(10)
uzel2 = Node(20)
uzel3 = Node(30)

uzel1.next = uzel2    # 10 ukazuje na 20
uzel2.next = uzel3    # 20 ukazuje na 30
# Máme: 10 → 20 → 30

Kompletní třída LinkedList

class LinkedList:
    def __init__(self):
        self.head = None  # Začátek seznamu

    def vypis(self):
        aktualni = self.head
        while aktualni is not None:
            print(aktualni.data, end=" → ")
            aktualni = aktualni.next
        print("None")

seznam = LinkedList()
seznam.head = Node(10)
seznam.head.next = Node(20)
seznam.head.next.next = Node(30)

seznam.vypis()  # 10 → 20 → 30 → None

2. Vkládání do spojového seznamu

A) Vložení na ZAČÁTEK (prepend) – O(1)

def vloz_na_zacatek(self, data):
    novy_uzel = Node(data)
    novy_uzel.next = self.head  # Nový ukazuje na starý head
    self.head = novy_uzel       # Nový uzel je nový head
Před: head → [10|→] → [20|×]

Krok 1: nový_uzel = Node(5)
        [5|?]

Krok 2: nový_uzel.next = head
        [5|→] → [10|→] → [20|×]

Krok 3: head = nový_uzel
  head → [5|→] → [10|→] → [20|×]

B) Vložení na KONEC (append) – O(n)

def vloz_na_konec(self, data):
    novy_uzel = Node(data)
    if self.head is None:
        self.head = novy_uzel
        return
    aktualni = self.head
    while aktualni.next is not None:
        aktualni = aktualni.next
    aktualni.next = novy_uzel

C) Vložení DOPROSTŘED (na konkrétní pozici) – O(n)

def vloz_na_pozici(self, data, pozice):
    if pozice == 0:
        self.vloz_na_zacatek(data)
        return
    novy_uzel = Node(data)
    aktualni = self.head
    for i in range(pozice - 1):
        if aktualni is None:
            return
        aktualni = aktualni.next
    novy_uzel.next = aktualni.next
    aktualni.next = novy_uzel

3. Výmaz ze spojového seznamu

A) Výmaz ze ZAČÁTKU – O(1)

def vymaz_zacatek(self):
    if self.head is None:
        return
    self.head = self.head.next  # Head je teď druhý uzel

B) Výmaz z KONCE – O(n)

def vymaz_konec(self):
    if self.head is None or self.head.next is None:
        self.head = None
        return
    aktualni = self.head
    while aktualni.next.next is not None:
        aktualni = aktualni.next
    aktualni.next = None

C) Výmaz KONKRÉTNÍ HODNOTY – O(n)

def vymaz_hodnotu(self, hodnota):
    if self.head is None:
        return
    if self.head.data == hodnota:
        self.head = self.head.next
        return
    aktualni = self.head
    while aktualni.next is not None:
        if aktualni.next.data == hodnota:
            aktualni.next = aktualni.next.next  # Přeskoč uzel
            return
        aktualni = aktualni.next
Před: head → [10|→] → [20|→] → [30|→] → [40|×]
                        ↑
                   chceme vymazat 20

Krok 1: Najdi uzel PŘED hodnotou 20 (je to uzel 10)
Krok 2: Přesměruj: 10.next = 20.next = 30

Po:   head → [10|→] → [30|→] → [40|×]
               (20 je nedostupný, uvolní se z paměti)

4. Vyhledávání ve spojovém seznamu – O(n)

def vyhledej(self, hodnota):
    aktualni = self.head
    pozice = 0
    while aktualni is not None:
        if aktualni.data == hodnota:
            return pozice
        aktualni = aktualni.next
        pozice += 1
    return -1
head → [10|→] → [20|→] → [30|→] → [40|×]
        |         |         |
    10==30?   20==30?   30==30? ANO! Vrát pozici 2

Kdy použít spojový seznam?

Vhodný:

  • Často vkládáš nebo mažeš na začátku
  • Nevíš předem velikost dat
  • Nepotřebuješ přímý přístup k prvkům podle indexu

Nevhodný:

  • Potřebuješ rychlý přístup na index (array[5])
  • Málo vkládáš/mažeš, hodně čteš
  • Paměť je omezená (spojový seznam má overhead – pointer navíc)

Binární strom

Co je to binární strom?

Strom = hierarchická struktura, kde každý uzel může mít maximálně 2 děti (levé a pravé).

Terminologie

           [50]        ← ROOT (kořen)
          /    \
       [30]    [70]    ← Vnitřní uzly
       /  \      \
    [20] [40]   [80]   ← LISTY (nemají děti)
  • Root (kořen): Nejvyšší uzel (50)
  • Parent (rodič): Uzel, který má děti
  • Child (dítě): Uzel pod rodičem
  • Leaf (list): Uzel bez dětí (20, 40, 80)
  • Výška stromu: Nejdelší cesta od kořene k listu

Struktura uzlu stromu

class TreeNode:
    def __init__(self, data):
        self.data  = data
        self.left  = None   # Levé dítě
        self.right = None   # Pravé dítě

root = TreeNode(50)
root.left  = TreeNode(30)
root.right = TreeNode(70)
root.left.left  = TreeNode(20)
root.left.right = TreeNode(40)
root.right.right = TreeNode(80)

5. Operace nad binárním stromem

A) Vkládání do BST (Binárního Vyhledávacího Stromu)

Pravidlo BST: Levý podstrom < rodič < pravý podstrom

def vloz_do_bst(root, data):
    if root is None:
        return TreeNode(data)
    if data < root.data:
        root.left  = vloz_do_bst(root.left,  data)
    else:
        root.right = vloz_do_bst(root.right, data)
    return root

Časová složitost vkládání

  • Vyvážený strom: O(log n) – půlíme prostor jako binární vyhledávání
  • Nevyvážený strom: O(n) – může degenerovat na spojový seznam

[20][40]   [80]               [30]

B) Vyhledávání v BST

def vyhledej_v_bst(root, hodnota):
    if root is None:
        return False
    if root.data == hodnota:
        return True
    if hodnota < root.data:
        return vyhledej_v_bst(root.left,  hodnota)
    else:
        return vyhledej_v_bst(root.right, hodnota)

C) Výmaz z BST – tři případy

Případ 1: Uzel je list (nemá děti) – prostě odstraníme.

Případ 2: Uzel má jedno dítě – nahradíme ho jeho dítětem.

Případ 3: Uzel má dvě děti – nahradíme nejmenší hodnotou z pravého podstromu.

def vymaz_z_bst(root, hodnota):
    if root is None:
        return None
    if hodnota < root.data:
        root.left  = vymaz_z_bst(root.left,  hodnota)
    elif hodnota > root.data:
        root.right = vymaz_z_bst(root.right, hodnota)
    else:
        # Případ 1: list
        if root.left is None and root.right is None:
            return None
        # Případ 2: jedno dítě
        if root.left  is None: return root.right
        if root.right is None: return root.left
        # Případ 3: dvě děti
        min_uzel   = najdi_minimum(root.right)
        root.data  = min_uzel.data
        root.right = vymaz_z_bst(root.right, min_uzel.data)
    return root

def najdi_minimum(root):
    while root.left is not None:
        root = root.left
    return root

6. Průchody stromem

A) In-order (LDR) – Levý → Data → Pravý

def inorder(root):
    if root is not None:
        inorder(root.left)
        print(root.data, end=" ")
        inorder(root.right)
       [50]
       /  \
     [30] [70]
     /  \    \
   [20][40]  [80]

In-order: 20 30 40 50 70 80  ← seřazené!

B) Pre-order (DLR) – Data → Levý → Pravý

def preorder(root):
    if root is not None:
        print(root.data, end=" ")
        preorder(root.left)
        preorder(root.right)
Pre-order: 50 30 20 40 70 80  ← rodič před dětmi

C) Post-order (LRD) – Levý → Pravý → Data

def postorder(root):
    if root is not None:
        postorder(root.left)
        postorder(root.right)
        print(root.data, end=" ")
Post-order: 20 40 30 80 70 50  ← děti před rodičem

Použití průchodů:

  • In-order: Získání seřazených dat z BST
  • Pre-order: Kopírování stromu, prefixová notace
  • Post-order: Mazání stromu (smažu nejdřív děti, pak rodiče)

Klíčové rozdíly

Spojový seznam: lineární struktura, každý uzel max 1 následník, rychlé operace na začátku.

Binární strom: hierarchická struktura, každý uzel max 2 děti, rychlé vyhledávání (pokud vyvážený).

Otázky ke zkoušce

1. Jaká je výhoda spojového seznamu oproti poli?

  • Vložení/výmaz na začátku je O(1); dynamická velikost bez realokace

2. Kdy je spojový seznam pomalejší než pole?

  • Přístup na index: O(n) vs O(1); pole má lepší cache lokalitu

3. Co je to BST a jaké má vlastnosti?

  • Levý podstrom < rodič < pravý podstrom; umožňuje rychlé vyhledávání O(log n)

4. Proč může BST degenerovat na O(n)?

  • Pokud vkládáme seřazená data, strom se stane nevyváženým (jako spojový seznam)

5. Jaké jsou průchody stromem a k čemu slouží?

  • In-order: seřazené hodnoty z BST
  • Pre-order: rodič před dětmi
  • Post-order: děti před rodičem (mazání stromu)

Praktické příklady

class Node:
    def __init__(self, data):
        self.data = data
        self.next = None

class LinkedList:
    def __init__(self):
        self.head = None

    def vloz_na_zacatek(self, data):
        novy = Node(data)
        novy.next = self.head
        self.head = novy

    def vypis(self):
        aktualni = self.head
        while aktualni:
            print(aktualni.data, end=" → ")
            aktualni = aktualni.next
        print("None")

seznam = LinkedList()
seznam.vloz_na_zacatek(30)
seznam.vloz_na_zacatek(20)
seznam.vloz_na_zacatek(10)
seznam.vypis()  # 10 → 20 → 30 → None
class TreeNode:
    def __init__(self, data):
        self.data  = data
        self.left  = None
        self.right = None

def vloz(root, data):
    if root is None:
        return TreeNode(data)
    if data < root.data:
        root.left  = vloz(root.left,  data)
    else:
        root.right = vloz(root.right, data)
    return root

def inorder(root):
    if root:
        inorder(root.left)
        print(root.data, end=" ")
        inorder(root.right)

root = None
for h in [50, 30, 70, 20, 40, 80]:
    root = vloz(root, h)

inorder(root)  # 20 30 40 50 70 80 (seřazené!)
